# Controle de Estudos

# Sobre o projeto
Este projeto foi criado com o objetivo de praticar o uso do Git e GitHub, registrando conteúdos estudados ao longo do tempo.

---

# Funcionalidades
- Registro de conteúdos estudados
- Organização por temas
- Controle simples de aprendizado

---

# Tecnologias utilizadas
- Git
- GitHub

---

# O que foi aprendido
- Criar repositório no GitHub
- Clonar projeto
- Criar e editar arquivos
- Realizar commits
- Criar e trabalhar com branches
- Abrir Pull Request
- Fazer merge
- Corrigir commits com `--amend`
- Utilizar `.gitignore`
